# ওষুধ বুঝে নাও

Streamlit app that explains a prescription or medicine-package image in simple Bangla using Gemini.

## Run

1. Copy `.streamlit/secrets.toml.example` to `.streamlit/secrets.toml`.
2. Put your Google AI Studio API key in `.streamlit/secrets.toml`.
3. Install and run:

```powershell
.\run.ps1
```
